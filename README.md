
# MyCIDP 微信小程序 #

* 本小程序的查询功能只适用于 防灾科技学院 在校生使用
* 微信直接搜索“MyCIDP” 或 扫描小程序码即可使用

## 主要功能 ##

###  一、教务处课表、成绩、学校校历查询 ###

* 通过教务处账号查询本学期课表，历年成绩，查看校历
